# online-chat-socket
Bu proje, Python'ın **Socket** kütüphanesi kullanılarak geliştirilmiş, sunucu-istemci (server-client) mimarisine dayalı bir mesajlaşma uygulamasıdır. Marmara Üniversitesi Bilgisayar Mühendisliği ders çalışmaları kapsamında hazırlanmıştır.

## 🚀 Özellikler
*   **TCP/IP Bağlantısı:** Sunucu ve istemci arasında güvenli veri iletimi sağlar.
*   **Anlık Mesajlaşma:** Gerçek zamanlı mesaj gönderimi ve alımı desteklenir.
*   **Soket Programlama:** Python `socket` modülü kullanılarak ağ protokolleri uygulanmıştır.

## 🛠️ Teknik Detaylar
*   **Dil:** Python
*   **Geliştirme Ortamı:** PyCharm
*   **Versiyon Kontrol:** Git & GitHub

## 📂 Proje İçeriği
*   `server.py`: Sunucu tarafını yöneten ve bağlantıları kabul eden dosya.
*   `client.py`: Sunucuya bağlanarak mesaj gönderen istemci dosyası.
*   `.gitignore`: Gereksiz klasörlerin (örn: `.venv`) yüklenmesini engeller.

## 📋 Çalıştırma Talimatları
1. Terminalde önce sunucuyu başlatın: `python server.py`
2. Ardından yeni bir terminalde istemciyi başlatın: `python client.py`
